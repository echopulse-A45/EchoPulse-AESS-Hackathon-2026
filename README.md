🚀 Project Overview

EchoPulse is an intelligent, low-power radar-based system for real-time flood monitoring and sustainable water management.

It combines FMCW radar, low-energy sensors, and Event-Driven Activation to:

- Reduce energy consumption by up to 75%
- Enable early flood detection (6–24 hours ahead)
- Support smart reuse of floodwater

💡 Key Innovations

- 🌱 Green Radar System using Event-Driven Activation
- 📡 FMCW Radar + AI Prediction (6–24 hours early warning)
- 💧 Smart Water Reuse System
  - pH + Turbidity + Temperature analysis
  - Automatic redirection for irrigation
- 🗺️ Interactive Risk Dashboard (Red / Orange / Yellow)
- 🔗 IoT Network of low-power radar nodes
